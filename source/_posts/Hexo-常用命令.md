### Hexo 常用命令说明

| 命令                      |       命令说明       |
| :----------------------- | :------------------: |
| $ hexo new "我的文章名称" |       新建文章       |
| $ hexo server             |       启动服务       |
| $ hexo generate           |     生成静态文件     |
| $ hexo deploy             |      hexo 部署       |
| $ hexo clean              | hexo生成静态文件清除 |
